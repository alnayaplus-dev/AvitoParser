## 🚀 VoltParser — сбор данных с Avito

Автоматическая выгрузка статистики, фраз и семантики из объявлений Avito в CSV.

### 🔧 Основные возможности

- **Статистика:** заголовок, описание, просмотры (всего/сегодня), категория, ссылка.
- **Фразы:** умное извлечение заголовков с разбивкой по запятым, удаление дубликатов.
- **Семантика:** частотность слов, очистка от стоп-слов (предлоги, «продам», «новый» и т.д.) и чисел.
- **Защита от бана:** случайные задержки, повторные попытки, скроллинг, паузы после 5–7 объявлений.
- **Интеграция с аналитикой:** полученные данные можно передавать в нейросеть для генерации ключевых фраз, которые затем использовать для анализа спроса.

### 📥 Выходные файлы

- `stats_full.csv` – полные данные по каждому объявлению.
- `analytic_phrases.csv` – список уникальных фраз.
- `semantic_words.csv` – слово + частота (от большего к меньшему).

### 🎯 Как пользоваться

1. Перейдите в расширения Chrome/Edge (edge://extensions/
2. Включите режим разработчика
    
    ![1.png](attachment:26032973-8ed0-4013-b2c7-2f2587d67ea9:1.png)
    
3. Перетажите VoltParse.crx в окно расширений
    
    ![image.png](attachment:5e0ba725-3261-4d0d-b0ac-0eb724f5049b:d8740937-9feb-4307-9fe9-b688782f7176.png)
    
4. Откройте любую страницу Avito со списком объявлений (поиск, категория, продавец).
5. Нажмите иконку расширения → появится панель.
6. Выберите режим: **📊 Статистика**, **📈 Фразы** или **🔍 Семантика**.
7. Дождитесь завершения или нажмите **⏹ Стоп**.
8. CSV-файл скачается автоматически.

- Запросы идут с вашего IP – не злоупотребляйте.
- При 100+ объявлениях процесс может занять несколько минут.

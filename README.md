### AvitoParser

## 🚀 Avito Парсер — расширение для сбора данных с объявлений

**Avito Парсер** — это мощный инструмент для автоматического сбора информации с карточек товаров на Avito. Позволяет за пару кликов выгрузить статистику, ключевые фразы и семантическое ядро прямо в CSV-файл.

### 🔧 Возможности

✅ **Сбор полной статистики**

- Заголовок
- Описание
- Количество просмотров (общее и за сегодня)
- Категория
- Ссылки на фото  **(В РАЗРАБОТКЕ)**
- Прямой URL объявления

✅ **Анализ фраз**

- Извлечение всех заголовков объявлений на странице
- Разбиение на составные части по разделителям `/ | , -`
- Удаление дубликатов
- Результат — список уникальных фраз для дальнейшего анализа спроса

✅ **Семантический анализ**

- Подсчёт частоты слов в заголовках (с очисткой от предлогов, союзов и мусорных фраз)
- Сортировка по популярности
- Помогает выделить самые важные ключевые слова для SEO и контекстной рекламы

✅ **Очистка текста от шума (В РАЗРАБОТКЕ)**

- Удаляются знаки препинания, лишние пробелы, спецсимволы
- Исключаются стоп-слова: предлоги (в, на, за), союзы (и, или), частицы (не, бы), местоимения (этот, тот) и т.д.
- В статистике и фразах стоп-слова остаются для сохранения смысла, в семантике — удаляются

✅ **Удобное управление**

- Панель инструментов прямо на странице Avito
- Кнопки с цветовой индикацией и эффектами наведения
- Возможность остановить парсинг в любой момент
- Статус-бар с номером обрабатываемого объявления

### 📥 Формат выгрузки

Все данные сохраняются в **CSV** (с разделителем точка с запятой, UTF-8 с BOM).

Поддерживается Excel, Google Sheets, LibreOffice.

- **stats_full.csv** — полная статистика по всем объявлениям
- **analytic.csv** — список фраз (по одному на строку)
- **semantic.csv** — слова + частота упоминаний

### 🎯 Как использовать

1. Установите расширение в Chrome (через `chrome://extensions` в режиме разработчика или загрузите упакованным `.crx`).
2. Перейдите на любую страницу **Avito** со списком объявлений (поиск, категория, профиль продавца).
3. Нажмите на иконку расширения в панели инструментов — появится круглая панель в правом верхнем углу.
4. Выберите нужный режим:
    - **📊 Статистика** — соберёт все данные по каждому объявлению (задержка 500 мс между запросами, чтобы не перегружать сервер).
    - **📈 Фразы** — выгрузит список заголовков и их частей.
    - **🔍 Семантика** — построит частотный словарь ключевых слов.
5. Дождитесь завершения или нажмите **⏹ Стоп**, чтобы прервать.
6. CSV-файл скачается автоматически.

### ⚠️ Важно

- Расширение работает только на [https://www.avito.ru/](https://www.avito.ru/**)
- Парсинг выполняется в клиентской части браузера — все запросы идут от вашего IP.
- Не используйте слишком маленькие задержки, чтобы избежать блокировки Avito (в коде уже установлены безопасные паузы).
- Для больших списков (100+ объявлений) процесс может занять несколько минут.

### 🛠 Технические детали

- Manifest V3
- Используется `fetch` + `DOMParser` для извлечения данных
- Очистка текста на основе регулярных выражений и словаря стоп-слов
- Возможность легко расширить список игнорируемых слов
